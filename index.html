Profile プロフィール

苫米地式コーチング認定コーチの西口皓輝です。
高校教師として教育現場に携わりながら、ゴール設定の重要性に気づき、コーチングを通じて人々が自らの可能性を最大限に引き出すサポートをしています。

クライアントに隠された本音や感情から、前進するためのカギを見つけ出します。

変革のプロセスにおいて、苫米地式コーチングの真髄である「現状の外側のゴール」を設定することを大切にしています。これにより、クライアントは自分の枠を超えた目標を追い求め、人生における新たな可能性に挑戦することができます。

私自身の経験や、教育、コーチングを掛け合わせることで、"心の底から変わりたいのに変われない"と願うあなたに対して、爆発的に変化するサポートを提供しています。

どんな状況においても自分の力で未来を切り開くことができるという確信を持ち、共にその道を歩んでいきたいと思っています。

あなたも本当の自分を取り戻し、新しい未来を共に創り上げていきませんか？

Vision ビジョン

私のビジョンは、人々が「本来の自分」を取り戻し、心から自由に生きることをサポートすることです。
社会的な枠組みや過去の経験によって制約を受けている多くの人々が、自分の可能性を信じ、未来を切り開く力を発揮できるようになることが私の使命です。

コーチングは、未来から流れる時間を元に、かつて感じたことのないワクワクや楽しさを取り戻すことができます。
あなたが本当に望む姿を明確にし、そこに向かって進む勇気を持てるよう、私は全力でサポートします。

私の願いは、あなたが自由に、自信を持って、自分らしい人生を歩むこと。その変化があなた自身を超えて、周りの人々、そして世界に良い影響を与えることです。

Service 提供内容

  • 苫米地式コーチングセッション

    あなたが本当に望む姿に向かって、言語抽象度に留まらず、非言語も用いてサポートします。

  • 個別相談・体験セッション

    まずは一度、お話してみませんか？あなたの課題や可能性を一緒に探ります。

Flow セッションの流れ

  1. 1. お申込み
    まずはLINEまたはメールでご連絡いただき、あなたの状況や目標をお聞きします。
  2. 2. 日程調整
    あなたのライフスタイルに合わせて、最適なセッションの日程を調整します。
  3. 3. 体験セッション
    初回の体験セッションでは、あなたの状態や目標を深く理解し、今後の方向性を一緒に考えます。
  4. 4. 継続的サポート
    その後は、あなたに合ったペースで継続的にコーチングを進めます。目標達成に向けての進捗を随時確認しながら進行します。
