Profile 自己紹介

高校教師を経たのちに、コーチとして活動しております。
現在も教育に携わりながらも、プロのコーチとしてクライアントの方々の未来の実現に携わっております。

Vision ビジョン

人は本来、自由で創造的な存在です。
でも社会の常識や過去の経験が、その本質を見えにくくしています。

コーチングは「見えなくなっていた本当の自分」に再会し、
子どもの頃のような楽しさとワクワクを取り戻すプロセスです。

私の願いは、あなたが「自分の人生を、心から楽しめる」状態になること。
そして、その変化が周囲へ、世界へと広がっていくことです。

Service 提供内容

  • 苫米地式コーチングセッション

    あなた自身の「現状の外側のゴール」を明確にし、自己変革をサポートします。

  • 教員・保護者向け支援

    教育現場にいる方々のために、指導と対話に役立つマインドセットを提供します。

  • 個別相談・体験セッション

    まずは一度、お話してみませんか？あなたの課題や可能性を一緒に探ります。

Flow セッションの流れ

  1. 1. お申込み
    フォームまたはLINEよりご連絡ください。
  2. 2. 日程調整
    ご希望に応じて、日程を調整いたします。
  3. 3. 体験セッション
    約30分の無料体験を通して、お悩みや方向性を確認します。
  4. 4. 継続コーチング開始
    ご希望があれば、正式なセッションを開始します。
